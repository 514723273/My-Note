# 2 JSX 本质是什么

JSX 其实是语法糖

开发环境会将 JSX 编译成 JS 代码（React.createElement(标签，属性，子节点)）（例如 render 里的 return (...) 都变成了 return React.createElement(...) 都被替换了）

JSX 的写法大大降低了学习成本和编码工作量

（另外 JSX 已经成为了一个独立的标准，可被其他项目使用
